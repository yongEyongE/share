### 스터디 : 알고리즘 스킬업
* 교재명 : 알고리즘 문제해결 전략 (총 2권), 구종만 저
* 참여자 : 김용태, 명혁, 문지희
  
### 규칙
* 발표는 주단위로 하고, 기본적으로 수요일에 한다
* 발표시간은 저녁식사 후(5:30)에 바로 시작하며 끝나는 시간은 제한이 없다
* 발표준비를 못했을때는 스타벅스커피를 산다
* 발표날짜를 변경할때는 1주 전에 미리 얘기한다
* 발표자끼리 합의하여 발표순서를 변경할 수 있다
* 발표자료는 무엇이든 괜찮으며 칠판을 사용해 된다
